# 시나리오 명세 (Android 14 / 카카오톡 10.6.5)

## A. 사진 보내기
### A-1. 사진 1장 보내기
1. 채팅방 열기
2. 하단 `+` 버튼
3. `사진` 메뉴
4. 사진 1장 선택
5. `전송(종이비행기)`
6. 채팅창 전송 확인

### A-2. 사진 여러장 보내기
1. 채팅방 열기
2. 하단 `+` 버튼
3. `사진` 메뉴
4. 사진 2장 이상 선택
5. 선택 개수 확인
6. `전송`
7. 채팅창 전송 확인

## B. 사진 받기
### B-1. 사진 1장 받기
1. 채팅방 열기
2. 사진 크게 보기
3. 우상단 `더보기(⋮)`
4. `저장하기`
5. 갤러리 확인

### B-2. 사진 여러장 받기
1. 사진 묶음 열기
2. 사진 보기 화면 이동
3. 우상단 `더보기(⋮)`
4. `전체 저장`
5. 갤러리 저장 확인
