# 제품 설계 (MVP)

## 화면 구조
1. 홈
2. 연습 목록(카테고리별)
3. 단계형 연습 화면
4. 완료 화면

## 사용자 플로우
1. 홈에서 카테고리 선택
2. 연습 항목 선택
3. 단계별 안내 진행
4. 완료 후 재학습 또는 목록 복귀

## 단계형 연습 UI 원칙
- 상단: 단계 진행률(n/N)
- 본문: 가상 카카오톡 예시 이미지 + 현재 행동 설명
- 하단: 이전/다음/처음부터/다시 보기

## 어르신 UX 기준
- 큰 버튼/큰 글씨
- 용어 단순화
- 한 단계에 한 행동
- 실패해도 복구 가능한 흐름

## 기준 고정 노출
- 앱 화면에서 항상 기준 환경 표시
  - Android 14
  - 카카오톡 10.6.5
