# 검증 기록

## 빌드 검증
- 웹 서버 실행: 200 OK 확인
- Android debug APK 빌드: 성공
- APK 설치: 성공 (adb install)
- 앱 실행: MainActivity 실행 확인

## 기능 검증
- 홈 4메뉴 노출 확인
- 카테고리별 연습 항목 노출 확인
- 단계 진행(이전/다음/처음부터) 동작 확인
- 완료 화면 및 재학습 동작 확인

## 오프라인 검증
- 로컬 자산 기반 화면 로드
- 서비스워커 등록
- 네트워크/API 비의존 구조 확인

## 알려진 참고사항
- Codex 실행 세션 종료 시, 세션에서 띄운 GUI 에뮬레이터가 같이 내려갈 수 있음
- 사용자 터미널에서 `./launch-interactive.sh` 실행 시 가장 안정적
