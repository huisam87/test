# 카톡 연습하기 프로젝트 개요

## 목적
고령 사용자 대상 오프라인 연습 앱을 만들어, 카카오톡의 사진/동영상 송수신 핵심 동작을 단계별로 반복 학습할 수 있게 한다.

## 기준 환경 (고정)
- OS: Android 14
- 앱 버전: 카카오톡 10.6.5

## 핵심 원칙
- 오프라인 전용
- 로그인 없음
- 실제 카카오톡 자동 조작 없음 (가이드/연습형)
- 한 화면 한 행동
- 큰 글자/큰 버튼 중심

## 제공 기능 (MVP)
- 사진 보내기: 1장, 여러장
- 사진 받기: 1장, 여러장
- 동영상 보내기: 1개, 여러개
- 동영상 받기: 1개, 여러개

## 결과물
- 웹 오프라인 연습 앱
- Android WebView 래퍼 APK
- 실행 자동화 스크립트
- 계획/설계/구현/검증 문서 세트
